Drive Control:

When coding a robot, you must start with the basics: drive control. The drive control provides the basic instructions for how the robot is driven, or what buttons should be clicked for each action. The code shown to the left shows the code for the drive control for one of the competitions. In this code, two types of driving are available : Arcade style or Tank drive. A button on the controller is assigned to each of these drives, and when clicked, the selected action is performed.
